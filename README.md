# demo-internal-module-node

## PUC - Módulos em Node.js

### Atividade Prática 1 - Unidade 2

Criar um modulo em node para executar as seguintes operações:
- a) 1 + 1
- b) -123 + 123123
- c) 8 * 0
- d) 1239123 * 12313
- e) 123 / -12
- f) 313123 / 0
- g) 10 - 12
- h) 10.05 - 0.25
- i) "xpto" * 0

### Executar

`npm start`
